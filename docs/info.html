Quotes

Quotes is Zikula module to enter and display quotes, reflections. Provides a block displaying a random quote (in given interval and content caching), and incorporates an interface for adding, editing and maintaining information.

1. Categorization is enabled in Quote block, so every instance of the block can associate with different category (categories).
2. Block content is cached for given interval, separate for every different instance of Quote block. With this:
- improved performance of whole web site;
- change displaying content not per every click of the user, but in given time interval.
3. There are block display filters: by hour of the day, day of the week, day of the month, month.
4. There is a "status" column of quotes (Active/Inactive).
